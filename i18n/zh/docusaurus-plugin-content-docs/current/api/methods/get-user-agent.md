# getUserAgent

## 说明

在发送验证码识别请求时，务必提供来自 Windows 操作系统的最新 User-Agent（UA）。

使用最新的 Windows User-Agent 有助于提高验证码识别准确率，并减少错误发生。

:::info 方法地址

```http
https://capmonster.cloud/api/useragent/actual
```

:::
