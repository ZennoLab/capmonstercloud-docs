## 如何获取 websiteKey

1. 打开触发 Prosopo 验证码的网站。

2. 进入 **开发者工具（DevTools）** → **Network**，刷新页面，找到加载验证码 `image` 的请求（例如 API 请求：`https://example.prosopo.io/v1/prosopo/provider/client/captcha/image`）。

3. 在 **Request Headers** 中找到参数 `Prosopo-Site-Key`，复制其值，并在创建请求时使用。

![](./images/prosopo/sitekey-prosopo.png)
