# Perguntas Frequentes (FAQ)

Esta seção fornece respostas para perguntas comuns sobre o CapMonster Cloud, incluindo tipos de captcha suportados, processo de reconhecimento, integração com API, uso da extensão do navegador e SDK, além de recomendações para uso otimizado do serviço.

:::info
Se você não encontrou a resposta que procurava, entre em contato com o [suporte](https://helpdesk.zennolab.com/conversation/new).
:::

---

## Geral

<details>
  <summary>O que é o CapMonster Cloud e para que ele é usado?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>
      <strong>CapMonster Cloud</strong> é um serviço baseado em nuvem para reconhecimento automático de vários tipos de captchas:  
      reCaptcha V2, reCaptcha V3, reCaptcha Enterprise V2, GeeTest, Amazon (AWS WAF), Cloudflare Turnstile, Cloudflare Challenge e outros.
    </p>
  </div>
</details>

<details>
  <summary>Como o CapMonster Cloud se diferencia de outros serviços de reconhecimento de captcha?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p><strong>Vantagens do CapMonster Cloud:</strong></p>
    <ul style={{ color: 'black' }}>
      <li><strong>Confiabilidade e escalabilidade:</strong> o sistema funciona de forma estável sob qualquer carga, lida com grandes volumes de tarefas, escala com o crescimento do projeto e fornece uma alta porcentagem de tokens válidos — um fator chave para contornar proteções com eficiência.</li>
      <li><strong>Suporte para todos os principais tipos de captcha:</strong> dos mais populares aos raros e personalizados.</li>
      <li><strong>Preços flexíveis e transparentes:</strong> adequados tanto para pequenas empresas quanto para projetos de grande escala.</li>
    </ul>
  </div>
</details>

<details>
  <summary>Quais tipos de captchas são suportados pelo CapMonster Cloud?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Tipos suportados:</p>
    <ul style={{ color: 'black' }}>
      <li>reCaptcha V2, reCaptcha V3, reCaptcha Enterprise V2</li>
      <li>GeeTest</li>
      <li>Amazon (AWS WAF)</li>
      <li>Cloudflare Turnstile/Challenge</li>
      <li>Imperva (Incapsula)</li>
      <li>Captcha de texto (ImageToTextTask), captchas complexos com imagens (BLS, Shein, etc.)</li>
      <li>Basilisk, TenDl, Prosopo, Binance, Temu</li>
      <li>Yidun (NECaptcha)</li>
      <li>MTCaptcha</li>
    </ul>
  </div>
</details>

<details>
  <summary>Vocês possuem um programa de afiliados para desenvolvedores?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Sim, o programa de afiliados permite ganhar até <strong>30%</strong> dos gastos dos usuários na resolução de captchas.</p>
  </div>
</details>

## Técnico

<details>
  <summary>O que devo fazer se houver um erro no status de reconhecimento?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Exemplos de erros:</p>
    <ul style={{ color: 'black' }}>
      <li><strong>ERROR_KEY_DOES_NOT_EXIST</strong> — Chave de API inválida ou inexistente. Verifique se está correta.</li>
      <li><strong>ERROR_ZERO_BALANCE</strong> — O saldo é zero. Recarregue para continuar.</li>
      <li><strong>ERROR_CAPTCHA_UNSOLVABLE</strong> — A solicitação pode estar incorreta, o tipo de captcha pode não ser suportado ou o captcha está corrompido/defeituoso. Recomendamos verificar os parâmetros e confirmar que o tipo de captcha é suportado pelo CapMonster Cloud.</li>
    </ul>
    <p>Consulte a <a href="https://docs.capmonster.cloud/pt-br/docs/api/api-errors" target="_blank" rel="noopener noreferrer">documentação de erros</a> ou entre em contato com o suporte.</p>
  </div>
</details>

<details>
  <summary>Qual é a velocidade média de reconhecimento para diferentes tipos de captcha?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Em média, entre <strong>1 e 30 segundos</strong>, dependendo do tipo de captcha.</p>
  </div>
</details>

<details>
  <summary>Quais são os principais limites de uso do serviço?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>
      Existe um limite de <strong>120 solicitações por tarefa</strong> para obter um resultado. Isso significa que, após criar uma tarefa pelo método <code>createTask</code>, você pode verificar periodicamente seu status usando <code>getTaskResult</code>, mas recomenda-se fazer isso em intervalos de no mínimo uma solicitação a cada 2–5 segundos.
    </p>
  </div>
</details>

<details>
  <summary>Como posso integrar o CapMonster Cloud com meu software ou bot?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Use sua <strong>chave de API</strong> para conectar. Também estão disponíveis <a href="http://docs.capmonster.cloud/pt-br/docs/getting-start/#exemplos-de-c%C3%B3digo" target="_blank" rel="noopener noreferrer">bibliotecas</a> para integrar a API do CapMonster Cloud ao seu código.</p>
  </div>
</details>

<details>
  <summary>Onde posso encontrar a documentação mais recente da API CapMonster Cloud?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Na seção <a href="https://docs.capmonster.cloud/pt-br/docs/category/api" target="_blank" rel="noopener noreferrer">API</a> do site.</p>
  </div>
</details>

<details>
  <summary>O CapMonster Cloud suporta resolver o ReCaptcha Enterprise?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>O serviço suporta <strong>ReCaptcha v2/v3</strong> e <strong>ReCaptcha Enterprise V2</strong> comuns.</p>
  </div>
</details>

<details>
  <summary>Como posso encontrar todos os parâmetros necessários do captcha para uma solicitação?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Os parâmetros podem ser obtidos através de:</p>
    <ul style={{ color: 'black' }}>
      <li><strong>Ferramentas de desenvolvedor</strong> do navegador.</li>
      <li><strong>Scripts de parsing</strong> (consulte a seção de tipos de captcha em <em>"Como encontrar todos os parâmetros necessários para criar uma tarefa"</em>).</li>
      <li><strong>Extensão CapMonster Cloud</strong> — fornece todos os parâmetros necessários para vários tipos de captchas (consulte a seção <a href="https://docs.capmonster.cloud/pt-br/docs/extension/extension-main#mapeamento-de-par%C3%A2metros-de-captcha" target="_blank" rel="noopener noreferrer">"Mapeamento de Parâmetros"</a> para detalhes).</li>
    </ul>
  </div>
</details>
