# Frequently Asked Questions (FAQ)

This section provides answers to common questions about CapMonster Cloud, including supported captcha types, recognition process, API integration, browser extension and SDK usage, as well as recommendations for optimal service usage.

:::info
If you didn’t find the answer you were looking for, please contact [support](https://helpdesk.zennolab.com/conversation/new).
:::

---

## General

<details>
  <summary>What is CapMonster Cloud and what is it used for?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>
      <strong>CapMonster Cloud</strong> is a cloud service that helps automatically recognize various captcha types:  
      reCaptcha V2, reCaptcha V3, reCaptcha v2 Enterprise, GeeTest, Amazon (AWS WAF), Cloudflare Turnstile, Cloudflare Challenge, and more.
    </p>
  </div>
</details>

<details>
  <summary>How does CapMonster Cloud differ from other captcha recognition services?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p><strong>Advantages of CapMonster Cloud:</strong></p>
    <ul style={{ color: 'black' }}>
      <li><strong>Reliability and scalability:</strong> the system works stably under any load, handles large volumes of tasks, and scales as your project grows.</li>
      <li><strong>Built-in proxies:</strong> in most cases, you won’t need to connect your own proxies.</li>
      <li><strong>No software installation required:</strong> all you need is an internet connection and your API key.</li>
      <li><strong>Support for all major captcha types:</strong> from popular to rare and non-standard types.</li>
      <li><strong>Flexible and transparent pricing:</strong> suitable for both small businesses and large-scale projects.</li>
    </ul>
  </div>
</details>

<details>
  <summary>Which captcha types are supported by CapMonster Cloud?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Supported types:</p>
    <ul style={{ color: 'black' }}>
      <li>reCaptcha V2, reCaptcha V3, reCaptcha Enterprise V2</li>
      <li>GeeTest</li>
      <li>DataDome</li>
      <li>Amazon (AWS WAF)</li>
      <li>Cloudflare Turnstile/Challenge</li>
      <li>Imperva (Incapsula)</li>
      <li>Text captcha (ImageToTextTask), complex image captchas (BLS, Shein, etc.)</li>
      <li>Basilisk, TenDl, Prosopo, Binance, Temu</li>
      <li>Yidun (NECaptcha)</li>
      <li>MTCaptcha</li>
    </ul>
  </div>
</details>

## Technical

<details>
  <summary>What should I do to fix an error in recognition status?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Example errors:</p>
    <ul style={{ color: 'black' }}>
      <li><strong>ERROR_KEY_DOES_NOT_EXIST</strong> — invalid or non-existent API key. Check your key.</li>
      <li><strong>ERROR_ZERO_BALANCE</strong> — balance is zero. Top up your balance to continue.</li>
      <li><strong>ERROR_CAPTCHA_UNSOLVABLE</strong> — the request may be incorrect, captcha type is not supported, or the captcha itself is damaged or displayed incorrectly. We recommend carefully checking your request parameters and ensuring the selected captcha type is supported by CapMonster Cloud.</li>
    </ul>
    <p>See the <a href="https://docs.capmonster.cloud/docs/api/api-errors" target="_blank" rel="noopener noreferrer">error documentation</a> or contact support.</p>
  </div>
</details>

<details>
  <summary>What is the average recognition speed for different captcha types?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>On average <strong>1 to 30 seconds</strong>, depending on the captcha type.</p>
  </div>
</details>

<details>
  <summary>What are the main usage limits for the service?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>
      There is a limit of <strong>120 requests per task</strong> to get the result. This means that after creating a task via <code>createTask</code>, you can periodically check its status using <code>getTaskResult</code>, but it is recommended not to do this more often than once every 2–5 seconds.
    </p>
  </div>
</details>

<details>
  <summary>How can I integrate CapMonster Cloud with my software or bot?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Use your <strong>API key</strong> to connect. There are also <a href="https://docs.capmonster.cloud/docs/getting-start#code-examples" target="_blank" rel="noopener noreferrer">libraries</a> available to integrate the CapMonster Cloud API into your code.</p>
  </div>
</details>

<details>
  <summary>Where can I find the latest CapMonster Cloud API documentation?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>In the <a href="https://docs.capmonster.cloud/docs/category/api" target="_blank" rel="noopener noreferrer">API</a> section on the website.</p>
  </div>
</details>

<details>
  <summary>Does CapMonster Cloud support ReCaptcha Enterprise?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>The service supports standard <strong>ReCaptcha v2/v3</strong> and <strong>ReCaptcha Enterprise V2</strong>.</p>
  </div>
</details>

<details>
  <summary>How can I find all the necessary captcha parameters for a request?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Parameters can be obtained through:</p>
    <ul style={{ color: 'black' }}>
      <li><strong>Browser developer tools</strong>.</li>
      <li><strong>Parser scripts</strong> (see details in the captcha types section under <em>"How to find all necessary parameters for task creation"</em>).</li>
      <li><strong>CapMonster Cloud Extension</strong> — provides all required parameters for various captcha types (see the <a href="https://docs.capmonster.cloud/docs/extension/extension-main#captcha-parameter-mapping" target="_blank" rel="noopener noreferrer">"Parameter Mapping"</a> section for details).</li>
    </ul>
  </div>
</details>
