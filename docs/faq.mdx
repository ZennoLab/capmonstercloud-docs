# Часто задаваемые вопросы (FAQ)

В этом разделе собраны ответы на часто задаваемые вопросы о работе с CapMonster Cloud, включая поддерживаемые типы капч, процесс их распознавания, интеграцию с API, использование расширения и SDK, а также рекомендации по оптимальному применению сервиса.

:::info
Если вы не нашли нужного ответа, обратитесь в [службу поддержки](https://helpdesk.zennolab.com/conversation/new).
:::

---

## Общие

<details>
  <summary>Что такое CapMonster Cloud и для чего он используется?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>
      <strong>CapMonster Cloud</strong> — это облачный сервис для автоматического распознавания различных типов капч:  
      reCaptcha V2, reCaptcha V3, reCaptcha v2 Enterprise, GeeTest, Amazon (AWS WAF), Cloudflare Turnstile, Cloudflare Challenge и др.
    </p>
  </div>
</details>

<details>
  <summary>Чем CapMonster Cloud отличается от других сервисов по распознаванию капч?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p><strong>Преимущества сервиса CapMonster Cloud:</strong></p>
    <ul style={{ color: 'black' }}>
      <li><strong>Надёжность и масштабируемость:</strong> система стабильно работает при любой нагрузке, справляется с большими объёмами задач, масштабируется под рост проекта и обеспечивает высокий процент валидных токенов — ключевой фактор для эффективного обхода защит.</li>
      <li><strong>Поддержка всех основных типов капч:</strong> от популярных до редких и нестандартных.</li>
      <li><strong>Гибкая и прозрачная ценовая политика:</strong> подходит как для малого бизнеса, так и для масштабных проектов.</li>
    </ul>
  </div>
</details>

<details>
  <summary>Какие типы капч поддерживает CapMonster Cloud?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Поддерживаются:</p>
    <ul style={{ color: 'black' }}>
      <li>reCaptcha V2, reCaptcha V3, reCaptcha Enterprise V2</li>
      <li>GeeTest</li>
      <li>Amazon (AWS WAF)</li>
      <li>Cloudflare Turnstile/Challenge</li>
      <li>Imperva (Incapsula)</li>
      <li>Текстовая капча (ImageToTextTask), сложные капчи с изображениями (BLS, Shein и др.)</li>
      <li>Basilisk, TenDl, Prosopo, Binance, Temu</li>
      <li>Yidun (NECaptcha)</li>
    </ul>
  </div>
</details>

<details>
  <summary>Есть ли у вас партнёрская программа для разработчиков?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Да, партнёрская программа позволяет зарабатывать до <strong>30%</strong> от трат пользователей на решение капч.</p>
  </div>
</details>

  <div style={{ fontWeight: 'normal' }}>

</details>

  <div style={{ fontWeight: 'normal' }}>

## Технические

<details>
  <summary>Что предпринять, чтобы устранить ошибку в статусе распознавания?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Примеры ошибок:</p>
    <ul style={{ color: 'black' }}>
      <li><strong>ERROR_KEY_DOES_NOT_EXIST</strong> — неверный или несуществующий API-ключ. Проверьте правильность ключа.</li>
      <li><strong>ERROR_ZERO_BALANCE</strong> — баланс равен нулю. Пополните баланс для продолжения.</li>
      <li><strong>ERROR_CAPTCHA_UNSOLVABLE</strong> — возможно, запрос составлен неверно, тип капчи не поддерживается сервисом или сама капча повреждена либо отображается некорректно. Рекомендуем внимательно проверить параметры запроса и убедиться, что выбранный тип капчи поддерживается CapMonster Cloud.</li>
    </ul>
    <p>Ознакомьтесь с <a href="https://docs.capmonster.cloud/ru/docs/api/api-errors" target="_blank" rel="noopener noreferrer">документацией по ошибкам</a> или обратитесь в поддержку.</p>
  </div>
</details>

<details>
  <summary>Какова средняя скорость распознавания различных типов капч?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>В среднем от <strong>1 до 30 секунд</strong>, в зависимости от типа капчи.</p>
  </div>
</details>

<details>
  <summary>Какие основные лимиты в использовании сервиса?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>
      Имеется ограничение — не более <strong>120 запросов к одной задаче</strong> для получения результата. Это значит, что после создания задачи через метод <code>createTask</code> вы можете периодически проверять её статус с помощью <code>getTaskResult</code>, но делать это рекомендуется с интервалом не чаще одного запроса каждые 2–5 секунд.
    </p>
  </div>
</details>

<details>
  <summary>Как интегрировать CapMonster Cloud с моим софтом или ботом?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Используйте <strong>API-ключ</strong> для подключения. Также доступны <a href="https://docs.capmonster.cloud/ru/docs/getting-start#%D0%BF%D1%80%D0%B8%D0%BC%D0%B5%D1%80%D1%8B-%D0%BA%D0%BE%D0%B4%D0%B0" target="_blank" rel="noopener noreferrer">библиотеки</a> для интеграции API CapMonster Cloud в ваш код.</p>
  </div>
</details>

<details>
  <summary>Где найти актуальную документацию по API CapMonster Cloud?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>В разделе <a href="https://docs.capmonster.cloud/ru/docs/category/api" target="_blank" rel="noopener noreferrer">API</a> на сайте.</p>
  </div>
</details>

<details>
  <summary>Поддерживает ли CapMonster Cloud решение ReCaptcha Enterprise?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Сервис поддерживает обычную <strong>ReCaptcha v2/v3</strong> и <strong>ReCaptcha Enterprise V2</strong>.</p>
  </div>
</details>

<details>
  <summary>Как найти все нужные параметры капчи для отправки запроса?</summary>
  <div style={{ fontWeight: 'normal' }}>
    <p>Параметры можно получить через:</p>
    <ul style={{ color: 'black' }}>
      <li><strong>Инструменты разработчика</strong> браузера.</li>
      <li><strong>Скрипты-парсеры</strong> (см. подробно в типах капч в разделе <em>"Как найти все нужные параметры для создания задачи на решение"</em>).</li>
      <li><strong>Расширение CapMonster Cloud</strong> — позволяет ознакомиться со всеми необходимыми параметрами для разных типов капч (подробнее в разделе <a href="https://docs.capmonster.cloud/ru/docs/extension/extension-main#%D0%BC%D0%B0%D0%BF%D0%BF%D0%B8%D0%BD%D0%B3-%D0%BE%D1%82%D0%BE%D0%B1%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5-%D0%BF%D0%B0%D1%80%D0%B0%D0%BC%D0%B5%D1%82%D1%80%D0%BE%D0%B2-%D0%BA%D0%B0%D0%BF%D1%87" target="_blank" rel="noopener noreferrer">"Маппинг (отображение) параметров капч"</a>).</li>
    </ul>
  </div>
</details>

</details>

<details>
  <summary>Есть ли возможность получить возврат средств?</summary>
  <div style={{ fontWeight: 'normal' }}>

  </div>
</details>

<details>
  <summary>Как пополнить баланс и где его можно посмотреть?</summary>
  <div style={{ fontWeight: 'normal' }}>

  </div>
</details>

<details>
  <summary>Есть ли у вас тестовый период или бесплатный тариф?</summary>
  <div style={{ fontWeight: 'normal' }}>

  </div>
</details>
